BrokenBarrierException(Cyclicbarrier)

    表示栅栏已经被破坏，破坏的原因可能是其中一个线程await()时被中断或者超时，或者是某一个线程在等待时执行reset()方法


ConcurrentModificationException (Thread)

    线程并发修改异常 Arraylist Hashmap 等，多个线程同时向线程不安全的集合添加值等时候会触发

OutOfMemoryError (Heap)

    JVM Heap 溢出异常

UnsupportedOperationException()

    给通过Arrays.asList 创造的list 执行add 或 remove 时会报错。这个ArrayList不是java.util包下的，而是java.util.Arrays.ArrayList.
