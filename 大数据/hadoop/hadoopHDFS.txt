HDFS
    hadoop distribute file system 

    HDFS定义：
        HDFS（Hadoop Distributed File System），它是一个文件系统，用于存储文件，通过目录树来定位文件；其次，它是分布式的，由
        很多服务器联合起来实现其功能，集群中的服务器有各自的角色。
    HDFS的使用场景：
        适合一次写入，多次读出的场景。一个文件经过创建、写入和关闭之后就不需要改变

    块大小默认128M

    shell操作：
        hadoop fs 或 hdfs fs


        上传：

            hadoop fs -moveFromLocal ./localname /HDFSdirname 将本地文件 剪切 粘贴到HDFS指定的文件夹上

            hadoop fs -copyFromLocal ./localname /HDFSdirname 将本地文件 复制 粘贴到HDFS指定的文件夹上

            hadoop fs -put ./localname /HDFSdirname 等同于 -copyFromLocal

            hadoop fs -appendToFile ./filename /HDFSdirname/HDFSfilename 将本地文件追加到指定HDFS文件末尾


        下载：

            hadoop fs -copyToLocal /HDFSname/HDFSfilename ./ 从HDFS指定文件夹下载到本地

            hadoop fs -get /HDFSname/HDFSfilename ./newname 从HDFS指定文件夹下载到本地并重新命名 等同 copyToLocal

        直接操作：

            hadoop fs -ls /HDFSfilename  查看指定HDFS文件夹下面的文件

            hadoop fs -cat /HDFSname/FileNmae 查看指定文件中的内容

            hadoop fs -mkdir /dirname 或 hdfs fs -mkdir /dirname  创建文件夹

            hadoop fs -cp /sanguo/shuguo.txt /jinguo 从HDFS的一个路径拷贝到HDFS的另一个路径

            hadoop fs -mv /sanguo/wuguo.txt /jinguo 在HDFS目录中移动文件

            hadoop fs -tail /jinguo/shuguo.txt 显示一个文件的末尾1kb的数据

            hadoop fs -rm /sanguo/shuguo.txt 删除文件或文件夹

            hadoop fs -rm -r /sanguo 递归删除目录及目录里面内容

            hadoop fs -du -s -h /jinguo 统计文件夹的大小信息

            hadoop fs -setrep 10 /jinguo/shuguo.txt 设置HDFS中文件的副本数量
